# Limpieza_de_datos
Limpiador de Datos Web

Esta es una aplicación de Streamlit que automatiza la limpieza de datos
extrayendo el dominio de las URLs y formateando la tabla según el estándar
requerido.

## Cómo usar:

1.  Sube tu archivo Excel o CSV.
2.  La aplicación detectará las columnas clave: 'Content URL', 'Domain Rating', 'Website Traffic'.
3.  Descarga el archivo limpio con las columnas ['Content URL', 'Domain Rating', 'Website Traffic', 'Email', 'WhatsApp'].
